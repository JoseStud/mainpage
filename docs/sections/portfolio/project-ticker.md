# Project Ticker

## Purpose

Holds the narrow marquee section that sits between the intro shelf and the project lineup.

## Important anchors and classes

- `.marquee-box`: styling hook for the ticker container.
- `<marquee>` with `scrollamount`, `onmouseover`, and `onmouseout`: preserves current motion and pause behavior.

## Edit this when

- The ticker copy changes.
- The portfolio page needs different inline marquee behavior without changing the surrounding section structure.
