# Project Lineup

## Purpose

Contains the main "Project shelf" listing of project cards, statuses, tags, summaries, and links.

## Important anchors and classes

- `.project-stack`: wraps the full lineup.
- `.project-block`: single project card wrapper.
- `.project-top`, `.project-line`, `.project-status`, `.tag-row`, `.project-summary`, `.project-links`: card layout and metadata hooks.

## Edit this when

- Adding, removing, or reordering projects in the shelf.
- Changing per-project names, statuses, tags, summaries, or links.
