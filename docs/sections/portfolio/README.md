# Portfolio Sections

This directory documents the section partials that make up the portfolio page.

## Sections

- `shelf-intro.section.html`: hero shelf intro and the `project-shelf` anchor used by site navigation.
- `project-ticker.section.html`: marquee strip with the portfolio page ticker copy.
- `project-lineup.section.html`: main project shelf listing with cards, tags, and links.
- `portal-split.section.html`: composite `portal-split` wrapper for the language list and workbench notes.

## Editing notes

- Keep `#project-shelf` unchanged because shared navigation and homepage links target it.
- Preserve existing class names such as `project-stack`, `project-block`, `note-list`, and `portal-split`; styles depend on them.
- Edit the smallest section doc below before changing structure so the partial stays aligned with its purpose.
