# Portal Split

## Purpose

Keeps the lower two-column `portal-split` block together so the "By language" list and "Workbench notes" remain one stable layout unit.

## Important anchors and classes

- `.portal-split`: shared wrapper for both lower boxes.
- `.note-list`: list styling used in both child sections.
- The child `.box` sections currently map to "By language" and "Workbench notes".

## Edit this when

- Either lower list changes and the paired layout should still stay together.
- The relationship between the language inventory and the workbench notes needs structural updates.
