# Logbook Preview

## Purpose

Owns the compact home-page preview of recent writing entries and the link back to the full blog archive.

## Important anchors and classes

- Archive link target: `blog.html#reading-desk`
- Layout classes: `compact-feed`, `compact-line`, `compact-bullet`, `compact-date`
- Heading pattern: `.box-heading` with `.rss-link`

## Edit when

Edit this partial when featured post titles, dates, or the archive link copy changes. Keep the compact list structure intact so the preview still renders as a tight feed rather than full article cards.
