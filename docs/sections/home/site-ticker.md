# Site Ticker

## Purpose

Owns the narrow marquee strip below the welcome block. It is a single decorative status line that sets tone without introducing new navigation structure.

## Important anchors and classes

- Wrapper class: `marquee-box`
- Accessibility label: `aria-label="Site ticker"`
- Interactive element: the inline `marquee` start-stop behavior on hover

## Edit when

Edit this partial when the ticker copy changes or if the ticker container markup needs to stay aligned with shared marquee styling. Keep the hover stop/start behavior unless the shared interaction model changes.
