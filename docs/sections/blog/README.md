# Blog Sections

This directory documents the section partials used by `site-src/pages/blog/page.config.mjs`.

## Sections

- `reading-desk-intro`: `/blog.log` intro with the `#reading-desk` anchor and archive map.
- `blog-ticker`: the marquee strip between the intro and entry list.
- `recent-entries`: the recent post cards that link back to `#reading-desk`.
- `draft-board-life-tracker`: the `portal-split` wrapper that keeps `#draft-board` and `#life-tracker` together.

Edit the smallest section that matches the content change, and keep the documented anchors and layout classes stable unless the page structure is intentionally being redesigned.
